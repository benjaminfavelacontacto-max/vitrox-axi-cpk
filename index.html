El Corazón del Cálculo (script.js) El CPK (C pk ​ ) mide la capacidad de un proceso. Necesitas las especificaciones: LSL (Límite Inferior), USL (Límite Superior), la Media (μ) y la Desviación Estándar (σ). C pk ​ =min( 3σ USL−μ ​ , 3σ μ−LSL ​ ) JavaScript function calcularCPK(datos, usl, lsl) { const n = datos.length; const media = datos.reduce((a, b) => a + b) / n; const varianza = datos.reduce((a, b) => a + Math.pow(b - media, 2), 0) / (n - 1); const desviacionEstandar = Math.sqrt(varianza); const cpu = (usl - media) / (3 * desviacionEstandar); const cpl = (media - lsl) / (3 * desviacionEstandar); return Math.min(cpu, cpl).toFixed(4); } 3. Obtener límites de los inputs const usl = parseFloat(document.getElementById('usl').value); const lsl = parseFloat(document.getElementById('lsl').value); // 2. Convertir CSV a un arreglo de números // Nota: Ajusta el delimitador si tu archivo usa ";" en lugar de "," const lineas = csvText.split(/\r?\n/); let datos = []; lineas.forEach(linea => { const valor = parseFloat(linea.trim()); if (!isNaN(valor)) { datos.push(valor); } }); if (datos.length < 2) { alert("No se encontraron suficientes datos numéricos en el archivo."); return; } // 3. FileReader: Escucha cuando seleccionas un archivo en tu navegador. Lee el archivo como texto plano sin necesidad de un servidor (todo ocurre en la computadora del usuario).
